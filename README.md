# GroupDNA — WhatsApp Group Chat Decoded 🧬💬

> **"Spotify Wrapped, but for your friend group."**

GroupDNA is a lightweight, pure-Python analytics engine that parses raw WhatsApp chat exports (`.txt`) and turns them into a polished, screenshot-ready terminal report. It deep-dives into your friend group's conversational dynamics, uncovering behavioral trends, temporal heatmaps, and custom personality archetypes.

---

## 🚀 Key Features

* **The Edge-Case Parser:** Seamlessly handles real-world messaging data hurdles including multi-line message continuations, deleted message stubs, omitted media attachments (`<Media omitted>`), and system broadcast messages.
* **Temporal Insights:** Automatically tracks your group's overall activity span, pinning down the absolute busiest day and hour.
* **NumPy Activity Heatmap:** Maps message frequencies across a 24-hour cycle by populating a custom $6 \times 24$ integer matrix, rendering a clean visual heatmap directly in the terminal using Unicode shading blocks.
* **Vocabulary Analysis:** Manually strips punctuation and screens text against an extensive custom stop-word dictionary to calculate the group's (and individuals') favorite words.
* **Response Dynamics & Silent Streaks:** Calculates individual average response speeds (safely capping overnight gaps at 6 hours) and uncovers who went off-the-grid the longest.
* **Exclusive Algorithmic Archetypes:** Normalizes behavioral metric scores to uniquely assign each participant a single, definitive personality trait—such as *The Spammer*, *The Night Owl*, *The Group Mom*, *The Comedian*, *The Ghost*, or *The Late-Night Philosopher*.

---

## 🛠️ The Challenge: Built Under Strict Constraints

To focus purely on algorithmic logic and core execution, this project was built from scratch following an intentional, strict architectural rulebook:

* 🚫 **No Pandas** (All data frames, aggregations, and tables built with native dicts/lists)
* 🚫 **No Matplotlib** (All visual charts and heatmaps rendered as text in the terminal)
* 🚫 **No Regex** (All cleaning, tokenization, and parsing executed via string manipulation)
* 🚫 **No collections.Counter** (All frequency metrics managed through custom dictionary counters)
* 🧩 **Approved Stack:** Pure Python Fundamentals + **NumPy** (exclusively utilized for matrix operations and multidimensional indexing)
